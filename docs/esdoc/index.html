    davidosomething.com for 2016

    Static generated site for davidosomething.com

    Stack

    Requires node >= 4.2.0 for ES6 (I recommend using nvm, for which nvm use (or nvm install) will read .nvmrc and use the version of node specified there. The package avn can auto-switch upon entering directory if installed).

    NOTE: g++ version 4.8 or higher is required for node-gyp compilation on node 4. This particularly applies to the Travis-CI deployment.

    Build

    Install dependencies:

    npm install --global gulp jspm
npm install
jspm install

    And run gulp to build.

    Note that jspm is installed globally for local dev, whereas the CI-deployment scripts all use npm run so the jspm bin is provided by the local node module.

    Deploy

    There are scripts configured in package.json to deploy via CI.
    Output is generated to public/ and, if running on master from Travis, deployed to surge.sh and my GitHub Pages repo.
